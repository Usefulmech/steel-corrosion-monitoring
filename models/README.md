# Model Weights

Model weights are not stored in Git due to size limitations.

## Download Instructions

After training on Google Colab, download:
- `detection_best.pt` - YOLOv12n detection model
- `classification_best.pt` - YOLOv8n-cls classification model

Place them in this directory.

## File Sizes
- detection_best.pt: ~6 MB
- classification_best.pt: ~3 MB
